📝 Project Description — iTask: Todo Manager

iTask is a modern and responsive Todo Management Web App built using React (Vite) and Tailwind CSS.
It helps users efficiently manage their daily tasks with a simple, fast, and elegant interface.

🚀 Features

✅ Add, edit, and delete todos instantly

☑️ Mark tasks as completed with a checkbox

👁️ Toggle view to show/hide completed tasks

💾 Persistent storage using LocalStorage (your tasks remain even after reloading or closing the browser)

⚡ Fast performance powered by Vite

🎨 Responsive UI styled with Tailwind CSS

🔄 Auto-sync with browser storage (no backend required)

🧠 Tech Stack

Frontend: React (Vite)

Styling: Tailwind CSS

Icons: React Icons

Storage: LocalStorage (Browser-based)
